How it scores

Two cumulative totals run the whole tour, both lower-is-better. Placement points add up each pick's actual stage finish. Rank points rank the drafted riders against each other on the day (winner-of-the-pool gets 1) and add those. Placement rewards consistent low finishes; rank rewards drafting the actual stage winner.
